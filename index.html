The Hot Guyde Companion

Github

The Hot Guyde Companion is a full stack application built to help readers track their progress on exercises from my book, The Hot Guyde. Current features include a journal and tools for tracking one's columns method, downward-arrow technique, and exposure therapy exercises. It is built with Ember.js, Handlebars.js, Bootstrap, Ruby on Rails, HTML, SCSS, and Javascript.

DuckCloud

Github

DuckCloud is a full-stack file storage and management system I helped develop as a member of the team "Squishy Ducks." Technologies employed include HTML, CSS, Javascript, JQuery, Mongoose, MongoDB, Express, Handlebars, Bootstrap, and Amazon S3.

Pathfinder Stat Block Lab

Github

The Pathfinder Stat Block Generator is a tool built for use with Paizo's Pathfinder RPG, a tabletop game based on Dungeons and Dragons. Game Masters can use this tool to reference and generate "stat blocks" for their in-game creatures in an easy-to-use and aesthetically intuitive way. The lab makes use of HTML, CSS, Javascript, JQuery, and Ruby on Rails.

Tic Tac Toe

Github

Tic Tac Toe was my first project developed at General Assembly. It is a front-end application built with HTML, CSS, Javascript, JQuery, and Bootstrap that allows the user to play the classic game and log results on the server.
